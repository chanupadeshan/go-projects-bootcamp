# 📦 Inventory Manager

A simple inventory system that teaches structs, maps, and the `ok` idiom for safe map access.

## Core Concepts
- `struct` — Grouping related data
- `map[string]Item{}` — Key/value storage
- `ok` pattern — Safe map lookup with existence check

## Running
```bash
go run main.go
```

## Features
- Add items to inventory
- Remove items
- Check stock levels
