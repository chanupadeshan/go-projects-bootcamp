# 🎯 Smart Guessing Game

A command-line guessing game that teaches the basics of Go: `package main`, `:=`, and `for` loops.

## Core Concepts
- `package main` — Entry point of a Go program
- `:=` — Short variable declaration with type inference
- `for` loop — Universal looping construct in Go

## Running
```bash
go run main.go
```

## How to Play
Guess a number between 1 and 100. The program will tell you if it's too high, too low, or correct!
